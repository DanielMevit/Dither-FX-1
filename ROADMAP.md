# Roadmap — Dither FX

Forward-looking ideas for the project. Priorities can shift with maintainer time and feedback.

## Near term

- Tighter performance on very large documents (cancellation / progress during long runs)
- Optional export of settings as shareable preset files (beyond `localStorage`)
- Additional dither algorithms and preset packs where they add clear value
- Small UX polish from real-world use (shortcuts, panel density, accessibility)

## Medium term

- Broader batch / automation scenarios (within UXP and Photoshop’s rules)
- Further vector-output options if the API and performance allow
- Documentation and examples for common print and pixel-art workflows

## How to contribute

Open an issue or pull request on the repository. For architecture and UXP constraints that agents and contributors should follow, see [`agent instructions/AGENTS.md`](agent%20instructions/AGENTS.md).
